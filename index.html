Variable in Typescript

Here are two method for declare the Variable.
1) var keyword
E.g: var num = 5;
2) let keyword
E.g: let num = 5;

Number in Typescript

There are many Types of Number in Typescript some of them are
  1. Number
    2. Example:
    let firstNum : number;
    firstNum = 12.75;
    document.write(firstNum);
     Result:
  2. String
    3. Example:
    let myName : string;
    myName = "Rabi Kr Yadav";
    document.write(myName);
     Result:
  3. Boolean
    4. Example:
    let myBool : boolean;
    myBool = true;
    document.write('The give Boolean Value is ' + myBool);
     Result:
  4. Any
    5. Example:
    let anyNumbType : any;
    anyNumbType = 'Learning Typescript';
    document.write('We are ' + anyNumbType);
     Result:
  5. Array String
    6. Example:
    let arrStr : string[];
    arrStr = ['hello', 'array','strings'];
    document.write(arrStr);
     Result:
  6. Array Number
    7. Example:
    let arrNum : number[];
    arrNum = [45600,45700,45800,46400,46704];
    console.log(arrNum);
    document.write('Some Postal Code of Janakpur Area is ' + arrNum);
     Result:

Type Assertion

Type assertion is using for tell the typescript compiler which type of variable is used for showing the (Intelligence)suggestion on the codeEditior
There are two way for this.
1) With 'angle-bracket'
Syntax:
let message; // here is only declare the variable not its type
message = "Assertion with angle-bracket";
(>string>message).length;
 2) As-Syntax
Syntax:
let msg; // here is only declare the variable not its type
msg = "Assertion use as a syntax";
(msg as string).length; // assertion with syntax
