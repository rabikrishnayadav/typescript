Classes

TypeScript is Object Oriented JavaScript
A class in terms of OOP is a blueprint for creating object.
A class encapsulates data for the object.
For creating classes use the 'class' keyword to declare a class.
Syntax:
class class_name{
class scope
}

A classs definition can include the following Fields.
- A field is any variable declare in a class
Fields represent data related to objects
- Constructor : Responsible for allocating memory for the objects of the class
- Functions : Functions represent actions an object can take
They are also at times referreed to as methods
These components put together are termed as the data members of the class.
Example:
class User{
// declare the feilds
name:string;
age: number;
address: string;
// this is called method because this function is inside the class
getName(){ // declared the method
}
// delcare the method for age
getAge(){
}
// declare the method for address
getAddress(){
}
}
